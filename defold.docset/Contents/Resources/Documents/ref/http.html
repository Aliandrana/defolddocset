http.request

Perform a HTTP request.

Parameters

url target url
method HTTP method, e.g. GET/PUT/POST/DELETE/...
callback response callback
[headers] optional lua-table with custom headers
[post_data] option data to send

Examples

Basic HTTP-GET request. The callback receives a table with the response in the fields status, payload (the data) and headers (a table). 

local function http_result(self, id, response)
    print(response.status)
    print(response.payload)
    print(response.headers)
end
function example(self)
    http.request("http://www.google.com", "GET", http_result)
end
